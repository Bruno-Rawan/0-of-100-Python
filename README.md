# 0-of-100-Python
# Day 3

Treasure Island Game (Desafio Python)
Este é um projeto de jogo de aventura em texto desenvolvido como parte do curso "100 Days of Code: The Complete Python Pro Boot Camp". O objetivo do jogador é tomar as decisões corretas para encontrar o tesouro escondido na ilha.

Melhorias Implementadas
Diferente da versão básica do desafio, adicionei funcionalidades que tornam o código mais robusto e profissional:

Lógica Aleatória: O caminho correto, a ação e a porta premiada são sorteados a cada nova partida usando o módulo random.

Validação de Entrada: Implementei loops while para validar os inputs do usuário, garantindo que o programa não encerre por erros de digitação.

Interface Limpa: Uso do módulo subprocess e platform para limpar o console a cada reinício, garantindo uma experiência de usuário (UX) superior e compatibilidade entre sistemas (Windows/Linux).

Loop de Jogo: Opção de reiniciar o jogo sem fechar o terminal.

Tecnologias Utilizadas
Python 3.12

Módulos Nativos: random, subprocess, platform
